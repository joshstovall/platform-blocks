The Avatar component displays user profile images, initials, or icons. Supports different sizes, colors, and can be grouped together in an AvatarGroup.

### Presence indicator
- Toggle `online` to show or hide the built-in `<Indicator />` badge.
- Override `indicatorColor` to map online/away/offline states to your palette.
