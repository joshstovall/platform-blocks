## Bug Fixes

### Charts: Pan/Zoom Gesture Reliability
Replaced `useState` with `useRef` for gesture-tracking state (`lastPan`, `pinchTracking`, `lastTapTime`) in `LineChart` and `ScatterChart`. React state updates are asynchronous and were causing stale reads inside `PanResponder` callbacks, leading to missed or jittery pan/pinch interactions. Refs provide synchronous access within the same gesture frame.

### Charts: Double-Tap Reset No Longer Triggers After Drag
`resetOnDoubleTap` now distinguishes taps from drags by measuring the distance between `onPanResponderGrant` and `onPanResponderRelease` locations. A gesture is only considered a "tap" if the pointer moved less than 10px, preventing accidental zoom resets during pan gestures.

### Charts: NaN Guard for Line/Scatter Coordinates
Added `Number.isFinite()` guards in `AnimatedPoint`, `AnimatedLineSeries`, `getDataDomain()`, and `dataToChartCoordinates()` to prevent `NaN` values from propagating into SVG attributes and crashing the render.

### Charts: Tooltip Nearest-Point Expansion Fix
Fixed `useTooltipAggregator` `findNearestPoint` so the expansion loop uses a frozen threshold instead of mutating `bestDataDx` during iteration, which could skip valid same-distance candidates.

### Charts: Tooltip Viewport Clamping
`ChartPopover` now accounts for estimated tooltip dimensions when clamping to viewport edges, preventing the tooltip from overflowing off-screen.

### Charts: Memory Leak — Cancel rAF on Unmount
`ChartInteractionProvider` now cancels any pending `requestAnimationFrame` callbacks on unmount, preventing `setState` on an unmounted component.

### Charts: Popover Stale Closure Fix
`ChartPopover`'s document-level `click` and `mousemove` listeners now read `interactionPointer` and `visible` from refs, fixing stale closures that caused the popover to remain stuck open or ignore outside clicks. Timeouts are also cleaned up on unmount.

### AppShell: Web Scroll Forwarding in Margins
On web, scrolling with the mouse cursor in the margins outside the `maxWidth` content area now correctly forwards wheel events to the inner `ScrollView`, fixing dead zones on wide viewports.

## Improvements

### Charts: Synchronous Initial Measurement
`RootOffsetCapture` (in both `ChartBase` and `ChartsProvider`) now uses `useLayoutEffect` for the initial DOM measurement, ensuring the root offset is available before the first paint instead of after an async `useEffect`.

### Charts: Adaptive Spatial Index Grid Size
`SpatialIndex` now automatically computes its grid cell size based on data distribution (targeting ~√n cells per axis) when no explicit `gridSize` is provided, improving nearest-point lookup performance across varying dataset scales.

### Charts: Export Internal Hooks
The following hooks are now exported from `@platform-blocks/charts` for advanced use cases:
- `useChartAnimation`
- `useChartData`
- `useDataDecimation`
- `useDomains`
- `useNearestPoint`
- `usePanZoom`
- `useStreamingData`
- `useTooltipAggregator`

### BrandIcon: New Instagram Icon
Added `instagram` brand icon with gradient fill support via SVG `linearGradient` defs.

### BrandIcon: Updated SoundCloud Icon
Replaced the low-fidelity SoundCloud icon with a higher-detail version using the correct brand proportions.
