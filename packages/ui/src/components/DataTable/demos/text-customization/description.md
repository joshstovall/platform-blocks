`headerTextProps` styles every column header `<Text>`. `cellTextProps` styles default-rendered cell text (cells without a custom `cell` renderer). Both accept the full `<Text>` API — `ff`, `weight`, `tracking`, `uppercase`, `size`, `colorVariant`, `style`.
