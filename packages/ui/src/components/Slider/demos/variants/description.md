The `variant` prop selects the slider's visual treatment. `default` is the standard track + filled-thumb look; `filled` thickens the track for an iOS-volume feel; `outline` swaps the inactive surface for a border; `minimal` is a hairline track for dense UIs; `segmented` divides the track at tick boundaries; `unstyled` strips all chrome so you can fully restyle via `trackStyle` / `thumbStyle`.
