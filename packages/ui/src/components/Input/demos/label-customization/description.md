`labelProps` and `descriptionProps` accept any `<Text>` props (style, weight, ff, colorVariant, size, etc.) and apply them to the label and description slots. The same pattern works on Checkbox, Switch, Radio, Select, TextArea, NumberInput, and PinInput.
