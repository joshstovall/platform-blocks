`placeholderTextColor` overrides the muted default. `startSectionProps` and `endSectionProps` accept any `<View>` props (including `style`) and apply them to the wrapper around your `startSection` / `endSection` content — useful for backgrounds, padding tweaks, or accessibility ids without forking the input.
