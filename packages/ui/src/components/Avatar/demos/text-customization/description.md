`fallbackProps`, `labelProps`, and `descriptionProps` accept any `<Text>` props (`ff`, `weight`, `tracking`, `uppercase`, `size`, `colorVariant`) and apply them to the initials, label, and description slots respectively without changing the avatar dimensions.
