`labelProps` applies `<Text>` props (`ff`, `weight`, `tracking`, `uppercase`, `size`, `colorVariant`, `style`) to every tab label in the strip, complementing the existing `textStyle` escape hatch with the rich shorthand props the rest of the field family already supports.
