`titleProps` accepts any `<Text>` props (`ff`, `weight`, `tracking`, `uppercase`, `size`, `colorVariant`, `style`) and applies them to the dialog header without changing the rest of the chrome. The same prop is also accepted by `openDialog({ titleProps })` for imperative dialogs.
